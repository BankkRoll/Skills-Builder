# Skills Directory

Pre-built Claude Code skills from popular documentation sites.

## Installation

```bash
npx skills add BankkRoll/Skills-Builder
```

Or install a specific skill:

```bash
npx skills add BankkRoll/Skills-Builder/skills/<skill-name>
```

## Telemetry

The skills CLI collects anonymous telemetry to rank skills. To opt out:

```bash
DISABLE_TELEMETRY=1 npx skills add BankkRoll/Skills-Builder
```
