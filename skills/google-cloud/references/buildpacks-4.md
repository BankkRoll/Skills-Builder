> Google Cloud's buildpacks stacks

# StacksStay organized with collectionsSave and categorize content based on your preferences.

A stack is made up of a Linux distribution version and system packages, such as
OpenSSL and curl. The base image stack will have an impact on the
available packages.

| Stack ID | Environment | Description |
| --- | --- | --- |
| google-24-full | Ubuntu 24.04 | Base image extending Ubuntu 24.04 with system packages for editing media and web automation |
| google-22 | Ubuntu 22.04 | Ubuntu 22.04 with minimal system packages |
| google-22-full | Ubuntu 22.04 | Base image extending Ubuntu 22.04 with system packages for editing media and web automation |
| google-18-full | Ubuntu 18.04 | Base image extending Ubuntu 18.04 with system packages for editing media and web automation |

This page lists the Ubuntu system packages included in the available stacks.
These packages are installed through `apt-get install` when a function is being
built.
