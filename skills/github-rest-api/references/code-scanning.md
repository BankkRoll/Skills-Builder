# REST API endpoints for code scanning

> Use the REST API to retrieve and update code scanning alerts from a repository.

## About code scanning

You can retrieve and update code scanning alerts from a repository. You can use the endpoints to create automated reports for the code scanning alerts in an organization or upload analysis results generated using offline code scanning tools. For more information, see [Scan code for vulnerabilities](https://docs.github.com/en/code-security/code-scanning).
