# REST API endpoints for gitignore

> Use the REST API to get .gitignore templates that can be used to ignore files and directories.

## About gitignore

When you create a new repository on GitHub via the API, you can specify a [.gitignore template](https://docs.github.com/en/get-started/git-basics/ignoring-files) to apply to the repository upon creation. You can use the REST API to get .gitignore templates from the GitHub [.gitignore repository](https://github.com/github/gitignore).
